= XML Path Like ID  

From the version 4.0.0 of the ESPD, a new identifier format is used to replace the UUID in the 
data structure of the Request or the Response for the various descriptive elements 
(criterion, group, subgroup, requirement, question, caption). The new identifier is structure-based and 
is called "XML Path Like ID".
